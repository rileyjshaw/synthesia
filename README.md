synthesia
=====

**_Synthesizer synesthesia_**

Turn MIDI notes into colors.

## Install

```bash
git clone https://github.com/rileyjshaw/synthesia.git
cd synthesia
npm i
```

## Use

Plug in a MIDI keyboard, then:

```bash
npm run start
```
